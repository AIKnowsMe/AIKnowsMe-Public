## 🧠 What is AIKnowsMe?

**AIKnowsMe** is a symbolic and cognitive profiling engine for deep introspection. It reconstructs **psychological structure** using symbolic language, decision-making patterns, and introspective essays. Unlike fixed personality quizzes, it dynamically interprets natural input across three stages:

- 🧩 Word Association Mapping
- 🧭 Scenario-Based Decision Profiling
- 📝 Introspective Essay Analysis

The system is built on **symbolic AI**, **Jungian archetypes**, **cognitive modeling**, **NLP**, and **data science**.

## 🔍 Profiling Structure

| Round | Function | Focus |
| --- | --- | --- |
| **1. Word Association** | Free response to symbolic word | Metaphors, emotional valence, reflex polarity |
| **2. Scenario Profiling** | Symbolic dilemmas | Role orientation, value prioritization |
| **3. Essay Projection** | Open question (free form) | Symbolism, depth, archetypal voice |

---

## ⚙️ Engine Modules

### 1. Lexical Reflex Analyzer

- **Input:** **Single-word associations** in response to symbolic prompts
- **Detects:** Metaphor class (e.g., war-like, growth, shadow), emotional polarity, symbolic tone
- **Outputs:** Resonance map, affective profile, metaphorical alignment

### 2. Scenario Cognitive Modeler

- **Input:** Abstract dilemmas
- **Detects:** Leadership tendencies, ambiguity preference, empathy vs logic
- **Outputs:** Role map, archetype trajectory

### 3. Introspective Essay Interpreter

- **Input:** Reflective user-written essay
- **Analyzes:** Coherence, symbolism, depth, archetypal voice
- **Outputs:** 20 KPIs, symbolic alignment score, narrative richness

# AIKnowsMe™ – Public Documentation
**AIKnowsMe™** is a symbolic-introspective AI framework that enables self-discovery through language, reflection, and structured profiling.

This public release includes:
- Name and concept: **AIKnowsMe™**
- Profiling structure based on three rounds:
  1. Word Association Snapshot
  2. Scenario-Based Profiling
  3. Reflective Essay
- Methodological foundation: NLP, Data Science, and psychoanalytic theory (Jung, MBTI, Big Five)
- Core principles: symbolic cognition, introspective modeling, and profile generation through interpretive AI
